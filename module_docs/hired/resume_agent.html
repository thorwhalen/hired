hired.resume_agent

Resume Generation System - Core Architecture

A hybrid LangChain/DSPy system for AI-powered resume creation with manual and autonomous operation modes.

Architecture follows supervisor-worker pattern with: - ResumeSession: Stateful conversation manager for manual mode - ResumeExpertAgent: Autonomous controller using ResumeSession as a tool - Specialized worker agents: expansion, distillation, matching, search
