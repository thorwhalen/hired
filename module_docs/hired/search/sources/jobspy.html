hired.search.sources.jobspy

JobSpy source adapter.

Wraps the python-jobspy package to search across multiple job boards (LinkedIn, Indeed, Glassdoor, ZipRecruiter, Google) concurrently.

class hired.search.sources.jobspy.JobSpySource(sites: List[str] | None = None, proxies: List[str] | None = None)[source]

JobSpy source implementation.

Uses python-jobspy to scrape jobs from multiple sources simultaneously. Does not require API keys, but may be subject to rate limiting.
