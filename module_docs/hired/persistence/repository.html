hired.persistence.repository

A small repository base that maps validated domain models to a key-value store.

A Repository wraps any MutableMapping (typically a dol file store from hired.persistence.base) and a pydantic model class, handling (de)serialization and validation so domain code deals in models, not dicts.

class hired.persistence.repository.Repository(store: MutableMapping, *, model: type[M] | None = None)[source]

Validated CRUD over a MutableMapping for a single pydantic model type.

Subclasses set model and key_of(). The store holds plain JSON-able dicts; models are validated on the way in and out.

key_of(item: M) → str[source]

Derive the storage key for an item (defaults to its id field).
